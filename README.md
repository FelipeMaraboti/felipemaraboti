Desenvolvedor focado na construção de **interfaces modernas, rápidas e orientadas à experiência do usuário**.

Trabalho com **React + TypeScript**, criando aplicações escaláveis, com integração a APIs, arquitetura bem definida e foco em performance e usabilidade.

Tenho experiência no desenvolvimento de **sistemas completos**, atuando desde a interface até a estrutura de comunicação com o back-end.
